Ejercicio 3 · Bucles
Tejiendo patrones triangulares

En Woven, el código se teje como una tela: cada hilo forma parte de un patrón mayor. Usarás bucles anidados para construir un triángulo de caracteres.

La función tejer(n) debe retornar un triángulo de n filas, donde la fila i contiene exactamente i rombos ◆.

Usa for i in 1..n para iterar y concatena con +. No olvides el salto de línea \n al final de cada fila.

◆
◆◆
◆◆◆
◆◆◆◆
◆◆◆◆◆
Python

Equivalente en Python del código Woven actual:

def tejer(n):
  patron = ""
  for i in range(1, n+1):
    for j in range(i):
      patron += "◆"
  patron += "\n"
  return patron

resultado = tejer(5)
print(resultado)
Java

Equivalente en Java del código Woven actual:

public static String tejer(int n) {
  String patron = "";
  for (int i=1; i<=n; i++) {
    for (int j=0; j<i; j++) {
      patron += "◆";
    }
    patron += "\n";
  }
  return patron;
}
